OR GATES

OR Gates take two inputs and if either one are True then the output will also be True
As shown in the table below
INPUTS OUTPUTS
0\OFF 0\OFF 0\OFF
1\ON 0\OFF 1\ON
0\OFF 1\ON 1\ON
1\ON 1\ON 1\ON
OR Gates are used when several inputs can result in an output such as the number keys above the key board or the number keys some keyboards have off to the side
